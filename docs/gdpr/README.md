# GDPR Compliance Documentation

This directory contains GDPR compliance documentation, templates, and checklists for Invent Alliance Limited.

## Documents

### [Third-Party Data Processing Agreements (DPA) Checklist](./third-party-dpas.md)
Checklists for evaluating and maintaining Data Processing Agreements with third-party service providers, including:
- SMTP email service provider
- Hosting provider
- Google Maps embed
- General DPA requirements

### [Data Protection Impact Assessment (DPIA) Template](./dpia.md)
Template for conducting Data Protection Impact Assessments (DPIAs) as required by GDPR Article 35, including:
- Risk assessment framework
- Mitigation measures
- Safeguards and security measures
- Examples for current processing activities
